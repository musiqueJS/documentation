# NoteEnum

## What is a note enum?

A note enum is an enum that represents a note.

## How to use a note enum?

A note enum can be used with the following code:

```javascript filename="index.js" {3} copy
import { NoteEnum } from 'musiquejs';

const noteEnum = NoteEnum['C'];
```

List of note enums:

- D

- F
